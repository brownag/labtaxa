# labtaxa: Fetch and Process USDA-NRCS-NCSS Lab Data Mart 'SQLite' Snapshot

Supports automatic download of 'Lab Data Mart'
(<https://ncsslabdatamart.sc.egov.usda.gov/>) database snapshots.
Applies patches to the cached database for testing and use in R.
